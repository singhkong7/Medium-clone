**Medium Blog Clone**


This project is a Medium.com-inspired blog platform built with Next.js. It allows users to read, write, and share articles in a clean, modern interface.

Features
📝 Create, edit, and delete blog posts
👤 User authentication (we are using Google and Github Sign in)
📰 View all articles and individual article pages
💬 Comment on articles
🔍 Search and filter articles
📱 Responsive design for mobile and desktop
Getting Started
First, install dependencies:
npm install

Then, run the development server:
npm run dev

Open http://localhost:3000 in your browser to see the app.

Project Structure
app - Main Next.js app directory (pages, layout, etc.)
components - Reusable React components (e.g., Header, ArticleList)
public - Static assets (images, icons)
Customization
Edit page.tsx to change the homepage.
Add new components in the components folder.
Update styles in globals.css.
